# Привет, я Киричек Мария!

### Студент, начинающий исследователь в области машинного обучения

В настоящее время работаю над проектом **Радиомика-MAS**, который посвящён медицинской диагностике с применением методов машинного обучения. Изучаю базы данных, **SQL**, продолжаю развиваться в ML.

Открыта для сотрудничества в проектах, связанных с **большими языковыми моделями (LLM)** и **обработкой естественного языка (NLP)**. Также ищу наставничество и поддержку в построении production-решений и более глубоком освоении машинного обучения.

Готова делиться опытом в области машинного обучения, анализа данных и работы с базами данных. Связаться со мной можно по электронной почте: **kirimusha@mail.ru**.
